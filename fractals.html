Koch Curve

My modifications are:

  • Randomly adding a number between 5 and -5 to the angle I'm turning/turning back
  • Changing the color based on the angle that I'm turning

  def new_koch(t,depth,length,angle):
    t.pd()
    if(depth==1):
        t.fd(length)
        if(angle>=60):
            t.color('blue')
        else:
            t.color('red')
    elif(depth>1):
        angle+=(random.randrange(-3,3))
        new_koch(t,depth-1,length,angle)

        t.rt(angle*2)

Tree

My modifications are:

  • Randomly turning the tree by it's angle*1.7
  • Setting the ends to red or blue and setting all the other branches to green or orange

  def new_tree(t, depth, size, angle):
    if(random.randrange(2)==1):
        angle*=1.8
    if depth == 0:
        if(random.randrange(2)==1):
            t.color('red')
        else:
            t.color('blue')
        t.fd(size)
        t.bk(size)
        if(random.randrange(2)==1):
            t.color('green')
        else:
            t.color('orange')
    else:
        t.fd(size)
        t.rt(angle)
        new_tree(t, depth-1, size, angle)
        t.lt(2 * angle)
        new_tree(t, depth-1, size, angle)
        t.rt(angle)
        t.bk(size)
